# GDG PUP Student Portfolio

A personal portfolio website built as a **demo project** for the **GDG PUP HTML & CSS Study Jam 1**. This project demonstrates modern web development practices using semantic HTML5 and CSS3.

## üöÄ Features

-   **Google Brand Colors**: Utilizes the iconic Google Blue, Red, Yellow, and Green for accents and borders.
-   **Interactive Elements**:
    -   Hover effects on cards and buttons.
    -   "Back to Top" button.
-   **Sections**:
    -   **Header**: Introduction with social links and a dynamic badge.
    -   **About Me**: Personal introduction with a mascot image.
    -   **Tech Stack**: Grid layout displaying skills (Frontend, Backend, Cloud, AI, etc.).
    -   **Featured Projects**: Showcase of projects with status indicators (Deployed, In Progress).
    -   **Interests**: A list of personal interests using flexbox.
    -   **Contact**: Contact information with styled icons.

## üìÇ File Structure

-   **`index.html`**: The main structure of the page.
    -   Uses semantic tags like `<header>`, `<section>`, `<main>`, and `<footer>`.
-   **`styles.css`**: The stylesheet for the application.
    -   **CSS Variables**: Defines colors (`--google-blue`, `--bg-body`) for easy theming.
    -   **Flexbox & Grid**: Used for layout management (e.g., `.skills-container`, `.projects-grid`).
    -   **Media Queries**: Ensures the site looks great on smaller devices.
